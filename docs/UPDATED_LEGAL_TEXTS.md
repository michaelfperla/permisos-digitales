# Updated Legal Texts for WhatsApp Compliance

## Privacy Policy Updates

### Replace Section 1.1 with:

```
1.1 DATOS PERSONALES ESPECÍFICOS DE WHATSAPP

Cuando usted opta por utilizar nuestro servicio de notificaciones por WhatsApp, recopilamos y tratamos los siguientes datos adicionales:

• Número de teléfono móvil asociado a WhatsApp
• Historial de conversaciones con nuestro servicio (almacenado temporalmente por 1 hora)
• Fecha y hora de los mensajes enviados y recibidos
• Estado de entrega de las notificaciones
• Fecha y método de consentimiento para el uso del servicio
• Dirección IP desde donde se otorgó el consentimiento

Estos datos se recopilan únicamente cuando usted proporciona voluntariamente su número de WhatsApp y acepta expresamente recibir notificaciones por este medio.

IMPORTANTE: Las conversaciones de WhatsApp se almacenan únicamente durante 1 hora para procesar su solicitud. Los registros de notificaciones enviadas se conservan por 90 días para fines de calidad del servicio.
```

### Replace Section 4.1 with:

```
4.1 GESTIÓN DE NOTIFICACIONES POR WHATSAPP

Consentimiento:
Al proporcionar su número de WhatsApp durante el registro o solicitud de permiso, usted otorga su consentimiento expreso para recibir notificaciones transaccionales relacionadas con su trámite. Este consentimiento es:
• Voluntario: El servicio de WhatsApp es opcional
• Específico: Solo para notificaciones sobre su permiso
• Informado: Usted conoce el tipo de mensajes que recibirá
• Revocable: Puede cancelarlo en cualquier momento

Para dar de baja las notificaciones de WhatsApp:
• Envíe cualquiera de estos comandos a nuestro WhatsApp: "STOP", "BAJA", "DETENER"
• La baja es inmediata y automática
• Recibirá un mensaje de confirmación
• Su número se agregará a nuestra lista de exclusión permanente
• Seguirá recibiendo notificaciones importantes por correo electrónico
• Puede reactivar el servicio desde su perfil en la plataforma web

Para reactivar las notificaciones:
• Ingrese a su perfil en https://permisosdigitales.com.mx/perfil
• Active las notificaciones de WhatsApp
• Proporcione su número en formato: 52XXXXXXXXXX

Retención de datos de WhatsApp:
• Conversaciones activas: 1 hora (en memoria temporal)
• Registros de notificaciones: 90 días
• Registros de consentimiento: 2 años (obligación legal)
• Datos de usuarios eliminados: Archivados por 5 años (obligación fiscal)
```

### Add New Section 10:

```
10. ELIMINACIÓN DE DATOS PERSONALES

Derecho de Supresión:
Además de los derechos ARCO, usted tiene derecho a solicitar la eliminación completa de su cuenta y datos personales.

Proceso de Eliminación:
1. Ingrese a su cuenta en https://permisosdigitales.com.mx
2. Vaya a Configuración > Eliminar Cuenta
3. Confirme su correo electrónico
4. Su cuenta será eliminada inmediatamente

Datos que se eliminarán:
• Información personal del perfil
• Historial de WhatsApp
• Datos de contacto
• Preferencias y configuraciones

Datos que se conservarán (por obligación legal):
• Facturas y comprobantes fiscales (5 años - SAT)
• Registros mínimos para prevención de fraude
• Permisos vigentes hasta su vencimiento

Alternativas:
• Exportar sus datos antes de eliminar: GET /api/user/data-export
• Desactivar solo WhatsApp sin eliminar la cuenta
• Solicitar eliminación parcial de datos específicos
```

## Terms and Conditions Updates

### Update Section 13.6:

```
13.6. Baja del Servicio
a) El Usuario puede darse de baja en cualquier momento enviando cualquiera de los siguientes comandos:
   - "STOP"
   - "BAJA" 
   - "DETENER"
   - "CANCELAR SUSCRIPCION"
   - "NO MAS MENSAJES"
b) La baja es procesada automáticamente e inmediatamente
c) Se enviará un único mensaje de confirmación de baja
d) Su número será agregado permanentemente a nuestra lista de exclusión
e) No se enviarán más mensajes después de la confirmación
f) Las notificaciones importantes continuarán por correo electrónico
g) Para reactivar, debe hacerlo desde la plataforma web

13.7. Lista de Exclusión Permanente
a) Los números dados de baja se mantienen en una lista de exclusión permanente
b) Esta lista previene el reenvío accidental de mensajes
c) La reactivación requiere consentimiento explícito a través de la web
d) La lista de exclusión se mantiene incluso si crea una nueva cuenta
```

### Add New Section 14:

```
14. PROTECCIÓN DE DATOS Y DERECHOS DEL USUARIO

14.1. Exportación de Datos
El Usuario tiene derecho a descargar todos sus datos en formato JSON:
a) Acceda a su perfil
b) Seleccione "Exportar mis datos"
c) Recibirá un archivo con toda su información
d) Incluye: perfil, permisos, pagos, notificaciones

14.2. Eliminación de Cuenta
El Usuario puede eliminar permanentemente su cuenta:
a) Esta acción es irreversible
b) Se eliminarán todos los datos personales
c) Se conservarán solo datos requeridos por ley
d) No podrá eliminar si tiene permisos activos
e) Recibirá confirmación por correo

14.3. Auditoría de Consentimiento
Mantenemos un registro auditable de:
a) Cuándo otorgó consentimiento para WhatsApp
b) Desde qué IP y dispositivo
c) Cambios en sus preferencias
d) Solicitudes de baja o eliminación

14.4. Retención de Datos
Períodos de conservación:
a) Conversaciones WhatsApp: 1 hora
b) Registros de notificaciones: 90 días
c) Auditoría de consentimiento: 2 años
d) Datos fiscales: 5 años (ley mexicana)
```

## Key Changes Summary

1. **Conversation Retention**: Changed from "90 días" to "1 hora" to match actual implementation
2. **Opt-Out Commands**: Added all supported commands (STOP, BAJA, DETENER, etc.)
3. **User Deletion**: Added self-service account deletion process
4. **Data Export**: Added right to export personal data
5. **Opt-Out List**: Clarified permanent exclusion list
6. **Reactivation**: Must be done through web platform, not WhatsApp
7. **Audit Trail**: Added transparency about consent tracking
8. **Legal Retention**: Clarified 5-year requirement for tax records
